SDG 3: Health & Well-being Website 

Overview 

This website focuses on SDG 3: Good Health & Well-being, promoting healthy lives and well-being for people of all ages. It provides information, tips, and a visual gallery to encourage a balanced and healthy lifestyle. 

Features 

Responsive Design: Works on both desktop and mobile devices. 
Hero Section: Eye-catching introduction with SDG 3 theme. 
Info Section: Highlights the importance of health and well-being. 
Tips Section: Slide-toggle tips for healthy living using jQuery. 
Gallery Section: Image carousel showcasing healthy activities with auto-slide and manual controls. 
About / Contact Section: Provides contact information and social media links. 
Clean and Neat Design: Color palette inspired by health and wellness, using soft greens and whites. 
Technologies Used 

HTML5 – for structure 
CSS3 – for styling and responsive design 
JavaScript – for image carousel functionality 
jQuery – for slide toggle animation and smooth scrolling 
How to Use 

Clone or download the repository. 
Open index.html in a web browser. 
Navigate using the menu or scroll to explore sections. 
Click "Show / Hide Tips" to view healthy living tips. 
Use the gallery arrows to navigate through images or let the auto-slideshow run.
